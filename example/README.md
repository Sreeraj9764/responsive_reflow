# responsive_reflow example

A minimal app showing `RrAdaptiveScaffold`, `RrResponsiveGrid`, `RrPageContent`,
and spacing tokens from `responsive_reflow`.

```sh
flutter run
```

Resize the window (desktop/web) or rotate a foldable to watch navigation reflow
from a bottom bar → navigation rail → full sidebar, and the grid recompute its
column count by available width.
